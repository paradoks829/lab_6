# lab_6
В данном коде реализованная игра в пятнашки через терминал.
Для запуска нужна любая IDE, которая поддерживает python.
Для передвижения необходимо использовать клавиши "w", "a","s","d" соответственно.

Пункты реализации игры:
1. Написать логику игры
	a) Реализовать проверку на "победный" сценарий
	b) Написать поиск пустого значения, для дальнейшего перемещения, а также написать проверку для входных данных для перемещения.
	c) Написать функцию передвижения и проверку, можем ли мы двигать в эту сторону
2. Написать функцию перемешивания двумерного массива, чтобы каждый раз игровое поле было новым.
